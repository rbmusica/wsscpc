Permette la configurazione personalizzata di prodotti WooCommerce con cambio dinamico di immagini (layer PNG) e prezzi. Specifico per chitarre elettriche e altro.

== Description ==

Questo plugin aggiunge una metabox ai prodotti WooCommerce permettendo all'amministratore di definire caratteristiche configurabili (es. colore, materiale, componenti) con opzioni associate. Ogni opzione può avere un adeguamento di prezzo, un'icona, e un layer immagine PNG.
Nel frontend, i clienti possono configurare il prodotto, vedendo l'immagine e il prezzo aggiornarsi dinamicamente.

Caratteristiche:
* Configurazione di caratteristiche e opzioni per prodotto.
* Adeguamento di prezzo per opzione.
* Icone per opzioni.
* Layer immagine (PNG) per impatto visivo.
* Possibilità di definire opzioni che cambiano l'immagine base del prodotto.
* Layout frontend responsive con immagine fissa e opzioni scorrevoli.
* Integrazione con carrello e ordini WooCommerce.
* Dipendenze semplici tra opzioni.

== Installation ==

1. Carica la cartella `wss-custom-product-configurator` nella directory `/wp-content/plugins/`.
2. Attiva il plugin attraverso il menu 'Plugin' in WordPress.
3. Vai alla pagina di modifica di un prodotto WooCommerce per iniziare a configurarlo.

== Frequently Asked Questions ==

= Come configuro un prodotto? =
Nella pagina di modifica del prodotto WooCommerce, troverai una nuova metabox chiamata "WSS Product Configurator". Da lì puoi aggiungere caratteristiche e opzioni.
